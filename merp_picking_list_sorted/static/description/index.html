mERP Picking List (Sorted) module

Sorts list of pack operations in picking report in accordance with available strategies:

  • sort by source locations in alphabetical order
  • sort by location removal strategy priority

Please, see mERP Outgoing Routing module for more details about strategies.

Notes

This module is designed to work only with "mERP Warehouse" application that can be downloaded from:

  1. https://merpapp.com/ website - version obtained from here is suitable for all types of organizations
  2. Google Play - has limitations, not possible to customize app by request

"mERP Warehouse" app work both on regular mobile phones with Android 4.1+ and with professional barcode scanners. For full list of supported devices check F.A.Q.
